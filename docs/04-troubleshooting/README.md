## 🎯 使用指南

**遇到问题时**：
1. 🔍 先查看 [00 - 常见问题] 寻找快速解决方案
2. 🔧 如果是构建相关问题，查看 [01 - 构建脚本详解]
3. 💡 使用诊断命令：`npm run verify:dynamic`

**预防问题**：
- 📋 按照部署文档的检查清单操作
- 🔄 定期运行清理命令：`pnpm clean`
- 📊 监控系统资源使用情况
